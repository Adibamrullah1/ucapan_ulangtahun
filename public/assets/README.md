# Photo Assets

Place your photos here for the gallery!

## Required Files:

- `1.jpg` - First photo
- `2.jpg` - Second photo
- `3.jpg` - Third photo
- `4.jpg` - Fourth photo
- `5.jpg` - Fifth photo
- `6.jpg` - Sixth photo

## Tips:

- Use high-quality photos (recommended: 800x800 to 1200x1200 px)
- Name them exactly as listed above
- Supported formats: .jpg, .jpeg, .png, .webp

The gallery will show placeholders for any missing images.
