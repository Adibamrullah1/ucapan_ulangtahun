# Birthday Audio Files

Place your birthday song here!

## Recommended:

- `birthday-song.mp3` - The main background music

## Tips:

1. Use a sweet, romantic instrumental or your special song
2. Keep file size reasonable (under 5MB recommended)
3. Make sure the audio is royalty-free or you have rights to use it

## Suggested Songs:

- "Happy Birthday" instrumental version
- Your favorite romantic song together
- A soft piano/acoustic version of a meaningful song
